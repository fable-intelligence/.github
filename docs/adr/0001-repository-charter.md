# ADR 0001: Repository Charter

## Status

Accepted

## Context

Fable is forming a multi-repo engineering organization. `.github` needs clear ownership from day one.

## Decision

This repository owns the scope described in `README.md` and `docs/charter.md`.

## Consequences

- Work outside this scope should move to the appropriate Fable repo.
- Cross-repo contracts must be documented before implementation hardens.
- Launch-critical risks must be visible in issues.
